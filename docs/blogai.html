STEP 1 — 코스 선택

A코스 — 원고 최적화

원고가 있음. 알고리즘에 맞게 재구성·편집이 필요한 사람.

사진 있음: 원고 + 사진 올리면 자동 배치
사진 없음: B코스로 연결

B코스 — 원고 + 이미지 생성

텍스트는 있는데 사진이 없는 사람.

카드뉴스 / AI 이미지 생성 / 인포그래픽 중 선택.
레퍼런스 이미지 업로드 → 스타일 DNA 추출 후 일관된 톤으로 생성

C코스 — 완전 자동

아무것도 없는 사람. 키워드 하나만 줘도 됨.

AI가 키워드 분석 → 제목·구성 제안 → 원고 생성 → 이미지 생성까지 전부 자동

STEP 3 — 인풋 수집 방법

텍스트·파일

  • 직접 붙여넣기
  • .txt / .docx 파일 업로드
  • 키워드 / 제목 / 주제만 입력
  • 참조 이미지 업로드

음성 녹음

  • 브라우저 내 음성 녹음
  • 자동 텍스트 변환 (STT)
  • 변환된 텍스트 편집 가능
  • 그대로 원고로 사용

사진 업로드

  • 최대 20장 한꺼번에 업로드
  • 글 단락에 자동 배분
  • 네이버 권장 사이즈 자동 리사이징
  • 파일명 SEO 최적화
STEP 4 — AI 추천 미리보기 (확정 전 항상 보여줌)

키워드 분석 결과

Google Trends — 상대 인기도 (0~100)
급상승 연관 검색어
지역별 관심도
경쟁 키워드 난이도 추정
AI 분석 — 웹서치 기반
상위 노출 글 구조 파악
롱테일 키워드 추천
메인·서브 키워드 구조 설계

제목 3가지 자동 생성

  • 25자 이내 자동 준수
  • 키워드 앞 배치 자동 적용
  • 클릭 한 번으로 선택

추천 글 구조

  • 도입(훅) → 본문 → 마무리 구성
  • 정보형·리스트형·스토리형·비교형
  • 수정 후 확정 가능
STEP 5 — AI 핵심 기능

글쓰기 엔진

  • 원고 전체 자동 생성 (C코스)
  • SEO 기준 재구성·편집 (A코스)
  • 내 글체 학습 후 반영
  • 키워드 밀도 실시간 체크 (3~5회)
  • 글자 수 실시간 카운터 (800자+)
  • 단락별 개별 수정 가능

이미지 생성 엔진

  • 레퍼런스 업로드 → 스타일 DNA 추출
  • 전 섹션 일관된 톤 유지
  • 단락별 이미지 프롬프트 자동 생성
  • 프롬프트 복사 → 외부 도구 사용 가능
  • 외부 생성 이미지 업로드 후 삽입
  • 카드뉴스 / 인포그래픽 / AI사진 선택

리서치 엔진

  • Google Trends 상대 인기도 연동
  • 급상승·연관 키워드 자동 수집
  • 경쟁 글 구조 분석 (웹서치 기반)
  • 카테고리별 FAQ 자동 생성
  • 시장 가격대·USP 분석 제안

SEO 체크 엔진

  • 제목 25자 이내 자동 검증
  • 이미지 수 확인 (6~13개 권장)
  • 소제목(H태그) 구조 점검
  • 이미지 파일명 최적화 안내
  • 종합 점수 표시 (100점 만점)
